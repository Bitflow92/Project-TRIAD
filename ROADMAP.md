# Project TRIAD Roadmap

## Vision

Project TRIAD aims to become a comprehensive, AI-powered endurance coaching platform designed specifically for multisport athletes. The platform combines structured training plans, strength programmes, performance analytics, recovery management and intelligent coaching into a single self-hosted application.

The project follows the philosophy:

> **Train smarter. Recover better. Race stronger.**

Current primary athlete:

- Rynier van der Watt
- Ironman 70.3 / Ironman athlete
- Two gym sessions per week
- Self-hosted infrastructure
- AI-first development philosophy

---

# Development Philosophy

Project TRIAD follows several key principles:

- GitHub is the single source of truth.
- VPS is deployment-only.
- Every release is version controlled.
- Docker-based deployment.
- Mobile-first design.
- Apple meets Garmin visual style.
- Conservative progression philosophy.
- Data ownership remains with the athlete.
