# Project TRIAD

Project TRIAD is a personal, self-hosted endurance coaching platform for strength training, workout logging, progress intelligence and future AI coaching.

## Branches

- `main` contains the public GitHub Pages website only.
- `develop` contains active platform development.
- GitHub Pages publishes from the `docs` folder on `main`.
