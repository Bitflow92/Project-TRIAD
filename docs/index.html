What is Project TRIAD?

Endurance coaching, owned by you.

Project TRIAD is a self-hosted endurance coaching platform built to support triathlon training, strength work, performance tracking, recovery and future AI coaching. It brings swim, bike, run and gym progress into one practical system for athletes who want clear feedback without surrendering their data.
